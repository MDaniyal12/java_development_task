# Robot Movement Simulator

Java program that simulates robot movement on a grid.

## Requirements

Java 8 or higher

## Project Structure

src/
 commands/
 robot/
 Main.java
 GuidedMain.java 

## Running the Program

### Simple Mode
Run:
java Main

Input example:
5 5
1 2 S
MRMLM

Output:
3 1 S

### Guided Mode
Run:
java GuidedMain
